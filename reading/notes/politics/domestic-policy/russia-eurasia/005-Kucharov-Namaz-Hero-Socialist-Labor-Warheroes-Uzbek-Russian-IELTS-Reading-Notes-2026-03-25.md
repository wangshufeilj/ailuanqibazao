### 第一部分：核心身份 (Core Identity)

🇷🇺 **Кучаров / Намаз Турсунович** / – **звеньевой** / колхоза имени Калинина / Ленинабадского района / Сурхандарьинской области / Узбекской ССР.
🇺🇸 **Kucharov / Namaz Tursunovich** / was a **team leader** / at the Kalinin **Collective Farm** / in the Leninabad District / of the Surkhandarya Region / in the Uzbek SSR.
🇨🇳 **库查罗夫 / 纳马兹·图尔苏诺维奇** / 曾是 / 乌兹别克苏维埃社会主义共和国 / 苏尔汉河州 / 列宁纳巴德区 / 以加里宁命名的**集体农庄**的 / **班组长**。

> 1. **Team leader** [tiːm ˈliːdə] `n.` **班组长**：在农业生产单位中负责一个小团队（звеньевой）的领导者。
> 2. **Collective farm** [kəˈlektɪv fɑːm] `n.` **集体农庄**：苏联时期的一种农业组织形式（колхоз）。
> 3. **District** [ˈdɪstrɪkt] `n.` **地区/区**：行政划分单位，此处对应俄语的район，雅思阅读常考行政地理词汇。
> 4. **Region** [ˈriːdʒən] `n.` **州/地区**：较大的行政区域，对应俄语的область。
>
> **注释**：
> *   **Uzbek SSR** (Uzbek Soviet Socialist Republic): 乌兹别克苏维埃社会主义共和国，前苏联的加盟共和国之一。
> *   **Kalinin** (Mikhail Kalinin): 米哈伊尔·加里宁，苏联早期领导人，许多农场和城市以其命名。

> 3. **Territory** [ˈterətri] `n.` **领土/地域**：指特定的地理区域。
> 4. **Nowadays** [ˈnaʊədeɪz] `adv.` **现今/如今**：用于对比过去与现在的状况。

🇷🇺 **Узбек**.
🇺🇸 **He was Uzbek**.
🇨🇳 **他是乌兹别克族**。

🇺🇸 His **education** / was at the **secondary level**.
🇨🇳 他的**学历** / 为**中学程度**。

🇷🇺 С 1963 года / **работал** / **хлопкоробом** / в колхозе имени Калинина / Ленинабадского района / (ныне – Сариасийского района).

### 第三部分：主要成就与荣誉 (Major Achievements & Honors)

🇷🇺 В **короткие сроки** / **в совершенстве** / **овладел** / профессией / и / **стал** / одним из / **передовых** / хлопкоробов / республики.
🇺🇸 In a **short period**, / he **perfectly** / **mastered** / the profession / and / **became** / one of the / **leading** / cotton growers / of the republic.

> 1. **Short period** [ʃɔːt ˈpɪəriəd] `n.` **短时期**：在描述职业晋升或技能习得时常用。
> 2. **Master** [ˈmɑːstə] `v.` **精通/掌握**：雅思写作中表达“掌握某种技能”的高级动词。
> 3. **Profession** [prəˈfeʃn] `n.` **职业**：指需要专门技能的长期工作。
> 4. **Leading** [ˈliːdɪŋ] `adj.` **领先的/主要的**：常用于描述行业佼佼者。

🇷🇺 **Вскоре** / его / **назначили** / звеньевым / **хлопководческого** / звена.
🇺🇸 **Soon** / he / **was appointed** / as the leader / of a **cotton-growing** / team.
🇨🇳 **不久后**，/ 他 / **被任命为** / 一个**棉花种植** / 小组的组长。

> 1. **Appoint** [əˈpɔɪnt] `v.` **任命**：雅思听力及阅读中常涉及的职位变动词汇。
> 2. **Leader** [ˈliːdə] `n.` **领导者/组长**：此处对应俄语的звеньевой。

🇷🇺 Из года в год / **звено** / Кучарова / **добивалось** / **высоких урожаев** / «белого золота».
🇺🇸 Year after year, / Kucharov's / **team** / **achieved** / **high yields** / of "white gold."
🇨🇳 年复一年，/ 库查罗夫的 / **小组** / 在“白金”（棉花）生产上 / **取得了** / **高产**。

> 1. **Year after year** [jɪər ˈɑːftə jɪə] `adv.` **年复一年地**：表达持续性的常用状语。
> 2. **Achieve** [əˈtʃiːv] `v.` **取得/实现**：写作高频词。
> 3. **Yield** [jiːld] `n.` **产量**：雅思农业或环境类文章的核心词汇，指单位面积或总体的产出。
> 4. **White gold** [waɪt ɡəʊld] `n.` **白金**：对棉花的形象比喻，地道的修辞表达。

🇷🇺 По **итогам** / девятой пятилетки / (1971–1975) / он / **был награждён** / орденом Ленина.
🇺🇸 According to the **results** / of the ninth five-year plan / (1971–1975), / he / **was awarded** / the Order of Lenin.
🇨🇳 根据 / 第九个五年计划 / (1971–1975) / 的**结果**，/ 他 / **被授予** / 列宁勋章。

> 1. **Result** [rɪˈzʌlt] `n.` **结果/成果**：指阶段性考核的总结。
> 2. **Five-year plan** [faɪv jɪə plæn] `n.` **五年计划**：计划经济国家的典型经济发展模式。
> 3. **Award** [əˈwɔːd] `v.` **授予/颁发**：雅思常考动词，注意被动语态用法。
>
> **注释**：
> *   **Order of Lenin** (Орден Ленина): 苏联的最高勋章，授予在社会主义建设中建立卓越功勋的个人或团体。

🇷🇺 **Указом** / Президиума Верховного Совета СССР / от 26 февраля 1981 года / за **выдающиеся успехи**, / **достигнутые** / в **выполнении** / планов / и / социалистических **обязательств** / по продаже / государству / в 1980 году / **рекордного** количества / хлопка-сырца / и других продуктов / земледелия, / Кучарову Намазу Турсуновичу / **присвоено звание** / Героя Социалистического Труда / с вручением / ордена Ленина / и золотой медали / «Серп и Молот».
🇺🇸 By the **Decree** / of the Presidium of the Supreme Soviet of the USSR / dated February 26, 1981, / for **outstanding success** / **attained** / in the **fulfillment** / of plans / and / socialist **obligations** / regarding the sale / to the state / in 1980 / of a **record** amount / of raw cotton / and other / agricultural products, / Namaz Tursunovich Kucharov / **was conferred the title** / of Hero of Socialist Labor / with the presentation / of the Order of Lenin / and the "Hammer and Sickle" gold medal.
🇨🇳 根据 / 苏联最高苏维埃主席团 / 1981年2月26日的 / **法令**，/ 为了表彰其 / 在**履行** / 计划 / 及 / 向国家销售 / 1980年 / **创纪录**数量的 / 原棉 / 及其他农产品 / 等社会主义**义务**方面 / 所**取得的** / **杰出成就**，/ 纳马兹·图尔苏诺维奇·库查罗夫 / **被授予** / “社会主义劳动英雄”**称号**，/ 并被颁发 / 列宁勋章 / 和“镰刀与锤子”金质奖章。

> 1. **Decree** [dɪˈkriː] `n.` **法令/政令**：政府或领导机构颁布的具有法律效力的命令。
> 2. **Outstanding success** [aʊtˈstændɪŋ səkˈses] `n.` **杰出成就**：雅思写作中形容卓越成果的高级词组。
> 3. **Fulfillment** [fʊlˈfɪlmənt] `n.` **履行/完成**：指完成任务或计划的过程。
> 4. **Obligation** [ˌɒblɪˈɡeɪʃn] `n.` **义务/职责**：法律或道德上必须做的事。
> 5. **Record** [ˈrekɔːd] `adj.` **创纪录的**：形容数量达到前所未有的水平。
> 6. **Confer the title** [kənˈfɜː ðə ˈtaɪtl] `v.` **授予称号**：正式的荣誉表达方式。
>
> **注释**：
> *   **Supreme Soviet of the USSR**: 苏联最高苏维埃，苏联的最高立法机构。
> *   **Hero of Socialist Labor**: 社会主义劳动英雄，苏联时期授予在经济和文化建设中做出突出贡献的人的最高荣誉称号。
> *   **Hammer and Sickle medal**: 镰刀与锤子奖章，社会主义劳动英雄的专属奖章，象征工农联盟。

# Security beefed up in Islamabad ahead of high-level US-Iran peace talks
**伊斯兰堡加强安保措施，迎接美伊高层和平谈判**

**来源**：Geo Television Network（巴基斯坦）  
**作者**：Ayaz Akbar Yousafzai（Geo News 驻伊斯兰堡记者）、Maryam Nawaz（Geo Web Desk 编辑）  
**日期**：2026-04-10

---

## 文章结构脉络

1. **核心事件**（Para 1–2）：巴基斯坦伊斯兰堡准备主办美伊高层和平谈判，旨在结束中东敌对行动。
2. **外交会晤**（Para 3, 7–9）：内政部长 Mohsin Naqvi 会见美国大使 Natalie Baker；确认美方代表团名单（JD Vance、Steve Witkoff、Jared Kushner）；美方赞赏巴基斯坦外交斡旋角色。
3. **冲突背景**（Para 4–6）：战争起因于 2026 年 2 月 28 日美以袭击导致伊朗最高领袖身亡；巴基斯坦斡旋两周休战，成为德黑兰与华盛顿之间的沟通渠道。
4. **安保与民生影响**（Para 10–20）：主要道路分流或封闭；红区周边政府学校考试推迟；交通警察全员上路引导公众。

## 逐句精读

### 标题

🔹**Security / beefed up / in Islamabad / ahead of / high-level / US-Iran peace talks**  
🔸**随着美伊高层和平谈判即将举行，伊斯兰堡的安保工作已全面加强。**

> **Beef up** (phrasal v.) /biːf ʌp/ **加强，充实**  
> 语域：非正式，但常用于新闻头条。  
> 拓展：常搭配 beef up security / police presence。同义词：strengthen, reinforce, augment。

> **High-level** (adj.) /ˌhaɪˈlev.əl/ **高层的，高级别的**  
> 语域：正式（政治／商务）。  
> 拓展：指涉及组织或政府中担任要职的人员。常见搭配：high-level meeting / talks / negotiations。

🔸**此举正值巴基斯坦准备主持美国和伊朗高级官员会谈，旨在结束中东地区的敌对行动。**

> **Be set to** (phrase) /bi set tuː/ **准备好做某事，预计将发生**  
> 语域：新闻报道常用。同义词：be poised to, be ready to。

> **Hostility** (n.) /hɒsˈtɪl.ə.ti/ **敌对行为，战争状态**（复数 hostilities）  
> 语域：正式（军事／外交）。  
> 拓展：常用复数表示"战争行动"。常见搭配：end / cease / commence hostilities。近义词：antagonism, enmity。

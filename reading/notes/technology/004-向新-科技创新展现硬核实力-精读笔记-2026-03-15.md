### 精读笔记正文

**载人登月任务取得重要阶段性突破，"洪荒70"高温超导托卡马克装置创全球商业聚变新纪录，宇树机器人再登春晚展现高超"武功"……**

> **【注释与解析】**
> *   **载人登月（Manned Lunar Landing）：** 中国计划在2030年前实现中国人首次登陆月球。2025-2026年期间，长征十号运载火箭、梦舟载人飞船、揽月着陆器等进入关键研制阶段。
> *   **"洪荒70"（HH70）：** 
>     *   **背景：** 由全球首家商业聚变能企业——能量奇点（Energy Singularity）研制。这是全球首台全高温超导托卡马克装置。
>     *   **解析：** 托卡马克（Tokamak）被形象地称为"人造太阳"。"洪荒70"的突破标志着中国在商业聚变能领域利用高温超导技术走在了世界前列。
> *   **宇树机器人（Unitree Robotics）：** 
>     *   **背景：** 杭州宇树科技有限公司，全球著名的机器人科技公司。其四足机器人（机械狗）频繁亮相大型文化活动。
>     *   **成语积累：** **亮点纷呈**（liàng diǎn fēn chéng），形容出彩的地方非常多，接连不断。近义词：精彩纷呈、群星闪烁。
>     *   **英文表达：** Hardcore strength（硬核实力）、Cutting-edge technology（前沿科技）。

**新年以来，我国科技创新成果亮点纷呈，展现出硬核实力与蓬勃生机，为"十五五"时期开好局、起好步筑牢了科技根基。**

> **【注释与解析】**
> *   **"十五五"时期：** 指2026年至2030年。文章背景设定在2026年初，正是"十五五"的开局之年。
> *   **硬核（Hardcore）：** 
>     *   **解析：** 原指一种力量感强、节奏激烈的音乐风格，后引申为形容某种事物"很有实力"、"过硬"、"核心"。
>     *   **易混辨析：** 与"硬性"不同，"硬核"侧重于技术含金量和核心竞争力。

**抢占科技发展制高点**

> **【注释与解析】**
> *   **制高点（Commanding Heights）：** 原指作战时必须争夺的最高地形，现比喻带动全局的关键位置。
> *   **金句积累：** 抢占科技发展制高点，挺起民族工业脊梁。

**"十四五"以来，面对纷繁复杂的国际形势和国内转型升级的发展要求，我国坚定实施创新驱动发展战略，持续加大关键核心技术攻关力度，抢占科技发展制高点。**

> **【注释与解析】**
> *   **创新驱动发展战略（Innovation-driven development strategy）：** 党的十八大提出，强调科技创新是提高社会生产力和综合国力的战略支撑。
> *   **关键核心技术（Key and core technologies）：** 俗称"卡脖子"技术。

**"大国重器"接连落地，不断刷新中国创新的新高度——**

> **【注释与解析】**
> *   **大国重器：** 指关系国家安全、国民经济命脉的重大技术装备。

**"嫦娥六号"实现人类首次月球背面采样返回，"天问二号"踏上小行星探测的"追星"之旅；福建舰劈波斩浪，国产大型邮轮扬帆远航；CR450动车组刷新中国速度，全球首座第四代核电站正式商运投产，"祖冲之三号"再次打破超导体系量子计算优越性世界纪录……**

> **【注释与解析】**
> *   **嫦娥六号/天问二号：** 
>     *   **地点：** 月球背面、小行星2016 HO3。
>     *   **背景：** 嫦娥六号实现了月背采样，这是航天史上零的突破。"天问二号"计划于2025年前后发射。
> *   **福建舰：** 中国第三艘航空母舰，也是中国首艘弹射型航空母舰。
> *   **CR450：** 
>     *   **解析：** "复兴号"动车组的升级版，试验速度可达450公里/小时。
> *   **"祖冲之三号"：** 
>     *   **背景：** 由中国科学技术大学潘建伟团队研制。
>     *   **量子计算优越性（Quantum Supremacy）：** 指量子计算机在处理特定任务时表现出远超传统计算机的能力。
> *   **词汇积累：** **劈波斩浪**（pī bō zhǎn làng），形容不畏艰险，勇往直前。

**这些"首次""首座"，不仅见证着我国科技实力从跟跑到并跑、再到部分领跑的历史性跨越，更推动我国产业发展不断向全球价值链中高端迈进。**

> **【注释与解析】**
> *   **跟跑、并跑、领跑（Following, running alongside, and leading）：** 形象描述科技水平与国际领先水平的差距演变。
> *   **价值链中高端（High end of the value chain）：** 意味着从提供原材料、组装向研发、设计、品牌等高利润环节转型。

**亮眼成绩单的背后，是我国持续稳定的创新投入与日益完善的创新体系。**

**"十四五"时期，全社会研发经费投入年均增长10%，2025年全社会研发经费投入强度提升至2.8%，首次超过经济合作与发展组织国家平均水平。**

> **【注释与解析】**
> *   **经济合作与发展组织（OECD）：** 
>     *   **背景：** 由发达国家组成的国际组织，其平均研发强度通常被视为国家现代化水平的重要参考。
>     *   **研发经费投入强度：** 研发经费与国内生产总值（GDP）之比。2.8%是一个里程碑式的数据。

**我国创新根基越扎越牢，发展底气愈发充沛。**

**战略科技力量逐步壮大，不断拓展中国创新的新厚度——**

> **【注释与解析】**
> *   **国家战略科技力量：** 包括国家实验室、全国重点实验室、国家科研机构、高水平研究型大学、科技领军企业。

**5年来，国家实验室体系建设稳步推进，全国重点实验室重组全面完成，国家科研机构、高水平研究型大学科研能力持续提升，科技领军企业培育步伐不断加快，体系化、多层次的战略科技力量矩阵基本成型。**

> **【注释与解析】**
> *   **全国重点实验室重组：** 2022年起，我国对原有的国家重点实验室进行系统性重组，以聚焦国家重大战略需求。

**如今，我国研发人员总量稳居世界首位，全球创新指数排名首次跻身全球前十，24个集群进入全球百强创新集群、数量位居世界之首，各类创新要素加速汇聚，全社会创新活力持续迸发。**

> **【注释与解析】**
> *   **全球创新指数（Global Innovation Index, GII）：** 由世界知识产权组织（WIPO）发布。跻身前十标志着中国正式进入创新型国家前列。
> *   **创新集群（Innovation Clusters）：** 如"深圳-香港-广州"集群，代表了区域创新的密集程度。

**全球创新中心总干事、中国社会经济系统分析研究会副理事长赵刚介绍，"十四五"以来，我国科技创新实力稳步增强，对经济社会发展的支撑引领作用愈发凸显，在全球产业竞争中逐步占据主动。**

> **【注释与解析】**
> *   **赵刚：** 著名科技外交与战略专家，长期研究全球创新体系。

**"'十五五'时期，我们将在党中央坚强领导下，锚定科技强国建设的战略目标，坚持'四个面向'，发挥新型举国体制优势，推进教育科技人才一体发展，促进科技创新和产业创新深度融合，营造世界一流的创新环境，全面提升科技创新能力，为建成科技强国奠定坚实基础。"科技部部长阴和俊表示。**

> **【注释与解析】**
> *   **"四个面向"：** 面向世界科技前沿、面向经济主战场、面向国家重大需求、面向人民生命健康。
> *   **新型举国体制（New-type whole-nation system）：** 强调在社会主义市场经济条件下，集中力量办大事，强化国家战略意志。
> *   **阴和俊：** 现任科学技术部部长。

**新质生产力加速形成**

> **【注释与解析】**
> *   **新质生产力（New Quality Productive Forces）：** 
>     *   **核心定义：** 创新起主导作用，摆脱传统经济增长方式、生产力发展路径，具有高科技、高效能、高质量特征，符合新发展理念的先进生产力质态。

**科技创新和产业创新是发展新质生产力的基本路径。**

**"十四五"以来，我国始终以科技创新引领现代化产业体系建设，科技创新与产业升级双向赋能，新质生产力成为拉动经济高质量发展的核心引擎。**

> **【注释与解析】**
> *   **双向赋能（Mutual empowerment）：** 科技为产业提供技术，产业为科技提供应用场景和资金，形成良性循环。

**我国积极推动传统产业向高端化、智能化、绿色化转型。**

**"京华号"超大直径盾构机刷新工程掘进纪录；遍布全国的"灯塔工厂"成为制造业转型升级的新标杆；自主研制的大型露天矿机器人运输装卸系统实现规模化应用，为全球矿业智能化升级提供"中国方案"。**

> **【注释与解析】**
> *   **"京华号"：** 我国首台16米级超大直径泥水平衡盾构机，用于北京东六环改造工程。
> *   **灯塔工厂（Lighthouse Factory）：** 
>     *   **背景：** 由世界经济论坛（WEF）评选，代表当今全球制造业领域智能制造和数字化的最高水平。
>     *   **中国方案（Chinese solution）：** 意指中国为解决全球性问题提供的智慧和路径。

**同时，我国在新一代信息技术、新能源等领域持续突破，引领新兴产业跑出发展"加速度"。**

**我国已建成约480万个5G基站，赋能千行百业数字化升级；光伏电池效率不断刷新世界纪录，风电、光伏新增装机量持续领跑全球；新能源汽车产销量连续11年保持全球第一，为全球低碳发展贡献"中国力量"。**

> **【注释与解析】**
> *   **5G基站数量：** 截至2026年，中国已建成全球最大规模的5G网络。
> *   **近义词：** **跑出"加速度"** —— 突飞猛进、一日千里。

**着眼长远，我国超前谋划布局未来产业，为培育新质生产力、开辟新赛道注入新动能。**

**在刚刚落幕的米兰冬奥会上，基于阿里云千问大模型打造的"国家奥委会AI助手"，为各国代表团提供母语化的赛务信息服务，大幅提升了赛事运转效率。**

> **【注释与解析】**
> *   **米兰冬奥会：** 即2026年米兰-科尔蒂纳丹佩佐冬季奥林匹克运动会。
> *   **通义千问（Tongyi Qianwen）：** 阿里巴巴打造的大型语言模型。
> *   **背景：** 此处体现了中国大模型技术（LLM）在国际顶级赛事中的实际应用。

**国际奥委会主席考文垂称赞，中国的人工智能技术让米兰冬奥会成为"史上最智能"的一届奥运会。**

> **【注释与解析】**
> *   **考文垂（Kirsty Coventry）：** 国际奥委会要员，前奥运冠军。此处反映了国际社会对中国AI技术的认可。

**2025年以来，我国全面实施"人工智能+"行动，带动高端制造、新兴消费、新业态新模式爆发式增长。2025年，我国人工智能企业数量超6000家。**

> **【注释与解析】**
> *   **"人工智能+"行动：** 类似于"互联网+"，旨在通过AI赋能各行各业。

**不止人工智能，更多前沿技术正加快照进现实。**

**人形机器人整机技术持续突破，在汽车制造、物流搬运、电力巡检等场景落地应用；脑机接口技术已能帮助截瘫患者站立行走，协助盲人恢复光感；国产脑起搏器实现全面商用，成为人口健康领域高质量发展新引擎。**

> **【注释与解析】**
> *   **脑机接口（Brain-Computer Interface, BCI）：** 在人脑与外部设备之间建立的直接连接。
> *   **脑起搏器（Deep Brain Stimulator, DBS）：** 一种植入式神经调控设备，常用于治疗帕金森病等。

**"当前国际形势不稳定因素增多，我国加速推动新旧动能转换，快速崛起的战略性新兴产业和未来产业，有效弥补了传统产业调整带来的增长缺口，成为稳定宏观经济大盘、提升产业链供应链韧性的核心支撑，为全球产业变革与经济复苏注入了源源不断的中国动能。"赵刚说。**

> **【注释与解析】**
> *   **新旧动能转换：** 淘汰落后产能（旧动能），发展高新技术产业（新动能）。
> *   **大盘（Macroeconomic base）：** 经济学术语，指宏观经济的整体形势。

**迈向高水平科技自立自强**

**"十五五"时期是我国建设科技强国的关键阶段。**

**阴和俊表示，要抓住新一轮科技革命和产业变革的历史机遇，统筹教育强国、科技强国、人才强国建设，提升国家创新体系整体效能，全面增强自主创新能力，加快高水平科技自立自强，引领发展新质生产力。**

> **【注释与解析】**
> *   **三位一体：** 党的二十大报告提出，"教育、科技、人才是全面建设社会主义现代化国家的基础性、战略性支撑"。

**基础研究是科技创新的总源头，是实现高水平科技自立自强的根基所在。**

**2025年，我国基础研究经费占比达7.08%，创历史新高，高水平国际期刊论文数量和国际专利申请量连续多年位居世界第一。**

> **【注释与解析】**
> *   **基础研究（Basic Research）：** 指为了获得关于现象和可观察事实的基本原理的新知识而进行的实验性或理论性研究，不以任何专门或特定的应用或使用为目的。
> *   **PCT专利申请：** 衡量国家创新能力的国际指标。

**中国科学技术发展战略研究院科技与产业发展研究所副所长苏楠建议，"十五五"时期，要坚持自由探索和目标导向"两条腿"走路，推动科学研究同国家战略需求紧密结合。**

> **【注释与解析】**
> *   **自由探索：** 基于科学家的兴趣。
> *   **目标导向：** 基于解决具体国家难题。

**面向人工智能、量子科技、脑科学等前沿领域，超前布局原创性、颠覆性研究方向；健全基础研究多元化投入体系，优化投入结构，稳步提高基础研究经费占全社会研发经费的比重；搭建更多企业开展基础研究的平台，落实企业投入基础研究研发费用加计扣除等优惠政策，适当提高扣除比例，扩大适用范围。**

> **【注释与解析】**
> *   **颠覆性技术（Disruptive Technology）：** 一种能够打破现有技术平衡，甚至取代现有主流技术的新技术。
> *   **研发费用加计扣除（Additional Deduction for R&D Expenses）：** 企业的税收优惠政策，投入研发的钱可以按更高比例抵扣税款。

**企业是创新的重要主体。截至2025年，我国累计培育科技和创新型中小企业超60万家、高新技术企业达50.4万家、专精特新中小企业超14万家，为产业链供应链稳定提供了重要支撑。**

> **【注释与解析】**
> *   **专精特新（Specialized, Refined, Differential and Innovational）：** 指具有"专业化、精细化、特色化、新颖化"特征的企业。

**"十五五"时期，我国将继续推动创新资源向企业集聚，支持企业牵头组建创新联合体、更多承担国家科技攻关任务，鼓励企业加大基础研究投入，促进创新链、产业链、资金链、人才链深度融合。**

> **【注释与解析】**
> *   **创新联合体：** 由领军企业牵头，联合高校、科研院所共同攻关。
> *   **四链融合：** 创新链、产业链、资金链、人才链。

**"同时，要进一步推动科技创新与产业创新深度融合，打通科技成果转化堵点，通过打造中试基地、概念验证中心、开放应用场景，让科技成果落脚于产业升级、经济高质量发展和民生获得感增强，切实赋能新质生产力发展。"赵刚认为。**

> **【注释与解析】**
> *   **堵点（Bottlenecks）：** 
>     *   **解析：** 指在过程中阻碍流动的环节。在科技领域，常指"实验室"到"生产线"之间的转化难问题。
>     *   **中试基地（Pilot Scale Base）：** 在科研成果正式产业化前，进行放大实验的场所。
>     *   **概念验证中心（Proof of Concept Center, PoC）：** 帮助早期科研成果跨越从"实验室"到"市场化"之间"死亡之谷"的组织平台。

**展望未来，中国创新必将以坚定的步伐持续迈进，在全面建设社会主义现代化国家的新征程上，不断书写新的辉煌。**

> **【解析】**
> *   **金句积累：** 
>     *   科技兴则民族兴，科技强则国家强。
>     *   在全面建设社会主义现代化国家的新征程上，书写科技强国新篇章。

# 文件索引

本索引为工作区所有已分类文件的快速查阅入口。内容主要来自互联网，支持多语言、多类型。每次完成分类或生成摘要后，请在此追加或更新对应条目。

---

## 索引说明

| 字段 | 说明 |
|------|------|
| 标题 | 文件/内容主题 |
| 类别 | 顶层分类（见下方目录） |
| 子类 | 主题（LCC）或格式子目录，如 history/china、notes |
| 日期 | 录入日期 (YYYY-MM-DD) |
| 路径 | 相对路径。**标题可 Ctrl+Click 跳转** |
| 摘要 | 多媒体类链接到 `generated/summaries/` 中的 Markdown 摘要 |

**维护说明**：新增、移动或重命名文件后须同步更新本索引；各子目录内序号连续无重号。
